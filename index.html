Crystals Collector

Welcome Crystals Collector! The game where your alignment with the source energy that creates worlds is put to the test!

The number you see below is the Target Number, which must be equaled in order to win.

The four crystals below the Target Number each contain a unique randomized value. Your spiritual alignment with the source energy indicates what those values actually are.

To win, you need to match the total of the crystals to the Target Number. You will lose if you go over the Target Number.

When you lose, the game will restart with a new Target Number and new values for each crystal.
